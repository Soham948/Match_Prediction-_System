# Match_Prediction-_System
IPL Match Prediction System using Machine Learning (Logistic Regression with ~80% accuracy) with an interactive web interface built using Flask &amp; Streamlit
🏏 IPL Match Prediction System
A Machine Learning-based web application that predicts the winner of an IPL match using historical match data. The model is built using Logistic Regression and achieves around 80% accuracy. The application is deployed using Flask and Streamlit for an interactive user experience.
🚀 Features
Predict IPL match winner in real-time
User-friendly web interface
Trained on historical IPL data
Multiple ML models tested (Logistic Regression, Random Forest, XGBoost, etc.)
Best model selected based on accuracy
Displays prediction results instantly
🧠 Machine Learning Models Used
Logistic Regression ✅ (Final Model – ~80% accuracy)
Random Forest
Decision Tree
XGBoost
AdaBoost
🛠️ Tech Stack
Programming Language: Python
Libraries: Pandas, NumPy, Scikit-learn, Matplotlib
Web Frameworks: Flask, Streamlit
Tools: Jupyter Notebook
📊 How It Works
Data Collection (IPL historical dataset)
Data Cleaning & Preprocessing
Feature Engineering
Model Training & Evaluation
Model Selection (based on accuracy)
Deployment using Flask & Streamlit
📈 Output
Predicts the winning team based on user inputs
Shows probability of winning
(Optional) Graph of predicted vs actual results
